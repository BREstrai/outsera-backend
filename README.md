# Outsera Backend

## Descrição

Este é um projeto Spring Boot que processa dados de filmes e calcula intervalos de prêmios para produtores.

## Pré-requisitos

- **Java 17** ou superior
- **Maven** instalado
- **Banco de Dados H2** (embutido)

## Como Rodar o Projeto

- Compile e execute o projeto:
    - ```mvn spring-boot:run```
    - O servidor estará disponível em http://localhost:8080.

- Como Rodar os Testes
    - ```mvn test```
